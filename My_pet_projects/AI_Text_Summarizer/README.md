# LLM Summarizer

Приложение для суммаризации документов с использованием различных LLM движков.

## Возможности

- 📄 Поддержка форматов: PDF, DOCX, TXT
- 🤖 Множество LLM движков: Ollama, OpenAI, Google Gemini, g4f
- 📦 Автоматическое определение локальных моделей Ollama
- ☁️ Скачивание моделей Ollama прямо из приложения
- 🚀 Автоматический запуск Ollama при необходимости
- 📊 Различные режимы суммаризации: bullet points, outline, executive summary
- 🎯 Специализированные стили: technical, legal, scientific
- 👁️ Встроенный просмотрщик PDF

## Установка

### 1. Установите зависимости

```bash
pip install PyQt5 python-dotenv openai google-generativeai g4f requests pypdf python-docx pymupdf
```

### 2. Установите Ollama (рекомендуется)

Скачайте и установите Ollama с официального сайта: https://ollama.ai

После установки Ollama будет автоматически запущена приложением при необходимости.

### 3. (Опционально) Настройте API ключи

Создайте файл `.env` в корне проекта:

```env
OPENAI_API_KEY=your_openai_key_here
GOOGLE_API_KEY=your_google_key_here
```

## Использование

### Запуск приложения

```bash
python main.py
```

### Работа с Ollama

1. При первом запуске приложение проверит, запущена ли Ollama
2. Если Ollama не запущена, приложение предложит запустить её автоматически
3. В списке моделей вы увидите:
   - 📦 **Локальные модели** - уже скачанные на ваш компьютер
   - ☁️ **Облачные модели** - доступные для скачивания

4. При выборе облачной модели (☁️) приложение предложит скачать её
5. Скачивание происходит в фоновом режиме с отображением прогресса

### Суммаризация документа

1. Нажмите **Open File** и выберите документ (PDF, DOCX или TXT)
2. Выберите движок (рекомендуется Ollama для начала)
3. Выберите модель из списка
4. Выберите режим суммаризации:
   - **bullet** - краткие пункты
   - **outline** - структурированный план
   - **executive** - executive summary
5. Выберите стиль:
   - **technical** - технический
   - **legal** - юридический
   - **scientific** - научный
6. Нажмите **Summarize**

## Рекомендуемые модели Ollama

Для начала работы рекомендуем:

- **llama3.2:latest** - быстрая и качественная модель (3GB)
- **mistral:latest** - хорошее качество, средний размер (4GB)
- **phi3:latest** - компактная модель от Microsoft (2GB)

Для более качественных результатов:

- **llama3.1:8b** - отличное качество (4.7GB)
- **qwen2.5:latest** - высокое качество (4.7GB)

## Устранение неполадок

### Ollama не запускается автоматически

Запустите Ollama вручную в терминале:

```bash
ollama serve
```

### Модель не найдена

Скачайте модель вручную:

```bash
ollama pull llama3.2
```

### Ошибка подключения к Ollama

Убедитесь, что Ollama запущена и доступна на `http://localhost:11434`
